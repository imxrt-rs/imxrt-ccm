Configure the UART clocks with default divider

When configure returns, all UART clock gates will be set to off. Use clock_gate to turn on UART clock gates.

pub fn configure_divider(&mut self, divider: u32)[src]

Configure the UART clocks with a clock divider.

The divider should be between [1, 64]. The function will treat a 0 as 1, and anything greater than 64 as 64.

When configure_divider returns, all UART clock gates will be set to off. Use clock_gate to turn on UART clock gates.
